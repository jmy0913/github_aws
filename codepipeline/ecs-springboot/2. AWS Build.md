# AWS CodeBuild
- AWS CodeBuild는 AWS에서 제공하는 완전관리형 빌드 서비스로,
소스 코드를 자동으로 컴파일하고, 테스트하고, 실행 파일로 패키징해주는 도구입니다.
- 쉽게 말해, 코드를 빌드하는 자동화 서버이며 Jenkins와 같은 CI 툴을 AWS에서 제공하는 형태입니다.
